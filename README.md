# XVM-LS ユーザーガイド

XVM-LS は、World of Warships の戦闘ログとリプレイを読み取り、戦闘中・戦闘後の情報を見やすく表示するデスクトップアプリです。

味方・敵チームの傾向、各プレイヤーの戦績、使用艦艇の戦績、自分の戦績推移を確認できます。開発者向けのツールではなく、WoWS をプレイしながら戦闘情報を把握するためのアプリです。

## リポジトリについて

このリポジトリでは現在、実行ファイルのリリース配布のみを行っています。
ソースコードは後日公開予定です。
最新版の実行ファイルは Releases ページからダウンロードしてください。

## できること

- WoWS の戦闘開始を自動検出し、参加プレイヤーと艦艇情報を表示します。
- Wargaming Public API から取得した戦績をもとに、味方・敵チームの平均勝率、平均ダメージ、平均 K/D を比較します。
- プレイヤーごとに、艦艇戦績と指定条件に合うプレイヤー戦績を一覧表示します。
- 戦闘終了後、リプレイから与ダメージ、被ダメージ、観測、潜在ダメージ、キル、XP などの結果情報を追加表示します。
- 過去の戦闘を Battle Logs から開き直せます。
- 自分の戦闘履歴から、勝率・ダメージなどの Stats Progression を確認できます。

## 必要なもの

- WoWS のリプレイ保存が有効になっていること
- インターネット接続
- Wargaming Application ID

Wargaming Application ID は、Wargaming Developer Portal でアプリケーションを作成して取得します。

https://developers.wargaming.net/

## 初回設定

初回起動時に Initial Setup が表示されます。以下を設定してください。

| 項目 | 説明 |
| --- | --- |
| Wargaming Application ID | 戦績取得に使う Wargaming API の Application ID です。 |
| Game Directory | World of Warships のインストール先フォルダです。 |
| Replay Directory | WoWS のリプレイ保存フォルダです。通常は Game Directory から自動設定されます。 |
| Max Full Battle Data Limit | 詳細な戦闘データを保持する件数です。古い詳細データは Stats Progression 用の最小データとしてアーカイブされます。 |
| Max Total Battle Data Limit | アーカイブ済みデータも含めた総保持件数です。これを超えた古いデータは削除されます。 |

設定を保存すると、アプリが再起動して監視を開始します。

## 普段の使い方

1. XVM-LS を起動します。
2. World of Warships を起動し、戦闘を開始します。
3. 戦闘ログが検出されると、XVM-LS に戦闘情報が自動で表示されます。
4. 戦闘終了後、リプレイが保存されると戦闘結果の詳細が追加されます。
5. 過去の戦闘は左側の Battle Logs から選択できます。

まだ戦闘データがない場合は、待機画面が表示されます。初回は WoWS で 1 戦完了するとデータが表示されます。

## 画面の見方

### Battle Logs

左側の戦闘履歴です。戦闘日時と使用艦艇が表示されます。

- 戦闘をクリックすると、その戦闘の詳細を表示します。
- 鍵アイコンで、古いデータ整理や削除対象から保護できます。
- ゴミ箱アイコンは、詳細データを削除して Stats Progression 用の最小データだけを残します。

### 戦闘ヘッダー

現在選択中の戦闘について、Tier、マップ、戦闘種別、戦闘時刻、自艦、勝敗を表示します。

- FILTERS: 表示に使う戦績条件を変更します。
- Refresh: 最新の戦績を再取得します。現在保存されている戦績データは上書きされます。
- Compact: 上部の比較表示をコンパクトにします。
- 履歴ナビゲーション: 前後の戦闘へ移動します。

### Team Comparison

味方チームと敵チームを比較します。

- 平均勝率
- 平均ダメージ
- 平均 K/D
- 勝率帯の分布

比較に使われる戦績は FILTERS の条件に従います。

### Player Stats

味方・敵それぞれのプレイヤー一覧です。

- プレイヤー名
- Tier、艦種、艦名
- 使用艦艇の戦闘数、勝率、平均ダメージ、生存率、キル、K/D
- FILTERS に合うプレイヤー戦績
- 戦闘終了後の与ダメージ、被ダメージ、観測、潜在ダメージ、キル、XP

戦績が取得できないプレイヤーや、条件に合う戦績がないプレイヤーは空欄または `-` で表示されます。

### Stats Progression

自分の戦績推移を確認する画面です。通常は直近 1 か月のランダム戦が初期条件です。

Detail Analysis から、以下の条件で絞り込みできます。

- 戦闘種別: Random、Ranked、Co-op
- 分隊: All、Solo、Div 2、Div 3
- Tier
- 日付範囲
- 艦種、国籍、艦艇
- 表示形式: Chart / List
- 集計基準: Session / Cumulative

List 表示では、該当する戦闘へ移動できる行があります。詳細データがアーカイブ済みの戦闘は、推移計算には使えますが、戦闘詳細へ移動できない場合があります。

## データの整理

左下の Battle Data Deletion から、条件に一致する戦闘データを整理できます。

- Archive Full Data: 戦闘の詳細データを削除し、Stats Progression 用の最小データは保持します。
- Delete Matching Data: アーカイブ済みデータも含めて完全に削除します。
- Protected: 鍵アイコンで保護された戦闘は整理対象から除外されます。

完全削除したデータはアプリ上から復元できません。迷う場合は Archive Full Data を使ってください。

## 設定の変更

左下の Settings から設定を変更できます。

- Wargaming Application ID
- Game Directory
- Replay Directory
- データ保持件数
- 表示スケール

Application ID、フォルダ、保持件数を変更した場合は、保存後に再起動が必要です。表示スケールは Auto が推奨です。画面サイズや DPI に合わない場合のみ Manual で調整してください。

## トラブルシューティング

### 戦闘が表示されない

- WoWS のリプレイ保存が有効になっているか確認してください。
- Replay Directory が実際の `replays` フォルダを指しているか確認してください。
- Game Directory が WoWS のインストール先になっているか確認してください。
- アプリを起動してから WoWS の戦闘を開始してください。

### 戦績が表示されない

- Wargaming Application ID が正しいか確認してください。
- インターネット接続を確認してください。
- Wargaming API 側で一時的に取得できない場合があります。
- FILTERS の条件を狭くしすぎていないか確認してください。

### 戦闘結果が増えない

- 戦闘終了後に `.wowsreplay` ファイルが保存されているか確認してください。
- Replay Directory が正しいか確認してください。
- 戦闘終了直後は反映まで少し時間がかかる場合があります。

### 初期化に失敗する

- Application ID、Game Directory、Replay Directory を確認してください。
- Replay Directory にアクセス権があるか確認してください。
- 必要に応じて Settings で設定し直してからアプリを再起動してください。

## 注意事項

- XVM-LS は Wargaming.net と提携していない非公式アプリです。
- 表示される戦績は Wargaming Public API とローカルの WoWS リプレイ・ゲームファイルに基づきます。
- API の仕様、通信状態、プレイヤーの非公開状態、戦闘種別によっては一部データが取得できない場合があります。
- 本アプリは戦術判断を補助するためのもので、勝敗やマッチング結果を保証するものではありません。
